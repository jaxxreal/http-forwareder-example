# HTTP-forwarder

## Installation

Use the package manager [yarn](https://yarnpkg.com/) in project root to install dependencies.

```bash
yarn
```

## Usage

```bash
yarn start
```

## Testing

```bash
yarn test
```

## Formatting

```bash
yarn format
```
